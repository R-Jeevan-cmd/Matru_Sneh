# Matru-Sneh Health App

Matru-Sneh Health is an Android-based maternal healthcare application designed for rural and semi-urban expectant mothers. The application provides offline-capable pregnancy monitoring tools and healthcare guidance using a simple and accessible interface.

## Features

- Baby Kick Counter with double-tap prevention
- Pregnancy Checkup & Vaccination Reminders
- Nutrition & Daily Food Intake Tracker
- Pregnancy Danger Signs Alert System
- Mood & Sleep Monitoring
- Vitals Tracking and Health Monitoring
- AI-Based Pregnancy Health Guide
- Offline Data Storage using Room Database
- Reminder Scheduling using WorkManager
- Kannada-friendly healthcare support

## Technologies Used

- Kotlin
- Android Studio
- XML
- Room Database
- WorkManager
- RecyclerView
- Material Design
- MVVM Architecture
- Google Gemini API

## Project Objective

The project aims to improve maternal healthcare accessibility in rural areas by providing a digital pregnancy guide that works even without high-speed internet connectivity.
